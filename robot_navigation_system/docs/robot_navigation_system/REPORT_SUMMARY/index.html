机器人自主导航系统 - 项目汇报

项目简介

基于深度Q网络（DQN）的机器人自主导航系统，通过激光雷达感知和强化学习实现智能路径规划与避障。

核心技术

技术栈

  • 深度学习: PyTorch 2.11
  • 强化学习: DQN算法
  • 环境感知: 激光雷达（360度扫描）
  • 可视化: Matplotlib

网络架构

输入(364维) → FC(256) → FC(128) → FC(64) → 输出(5维)

激光雷达数据：360度环境感知

  9. 导航性能

  10. 机器人能够避开障碍物

性能指标

指标 值
状态维度 364（360激光雷达 + 4机器人状态）
动作空间 5（前进、左转前进、右转前进、原地左转、原地右转）
训练轮数 500
经验池大小 100,000
学习率 0.001
折扣因子 0.99
